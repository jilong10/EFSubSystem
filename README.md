# JellyBean - EF Subsystem

JellyBean deploys units and send status update to CMO

### Prerequisites

What things you need to install the software 

```
Node.JS
NPM
```

### Installing

To run the JellyBean

```
Go to the directory that this project is located
npm install
node index.js
```
